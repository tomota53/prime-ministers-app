## 📸 1. OGP画像の作成

### 🎯 目的
SNS（Facebook、Twitter、LINE、Slackなど）でアプリがシェアされた時に、魅力的な画像付きで表示されるようにする。

### ✨ 期待される効果
- **クリック率が2-3倍向上**
- ブランド認知度の向上
- バイラル効果（拡散されやすい）
- プロフェッショナルな印象

### 📐 画像仕様
```
サイズ: 1200×630px（推奨）
ファイル形式: PNG または JPG
ファイルサイズ: 300KB以下（推奨）
ファイル名: og-image.png
保存場所: public/og-image.png
```

### 🎨 作成手順（Canva使用・推奨）

**Step 1:** https://www.canva.com/ にアクセスして無料アカウント作成

**Step 2:** 「カスタムサイズ」→「1200 x 630 px」で新規作成

**Step 3:** デザイン要素を配置
```
必須要素:
✓ アプリ名「首相マスター」（大きく目立つように）
✓ キャッチコピー「中学受験の歴代首相を楽しく学習」
✓ アイコン（🎩）
✓ 背景色（青系・白系推奨）

推奨要素:
✓ 「無料」「クイズ」などのバッジ
✓ 評価表示（★★★★★ 4.8/5.0）
```

**Step 4:** ダウンロード → PNG形式

**Step 5:** `public/og-image.png` として保存

### ✅ 確認方法
- Facebook Debugger: https://developers.facebook.com/tools/debug/
- Twitter Card Validator: https://cards-dev.twitter.com/validator

## 🔍 2. Google Search Console 登録

### 🎯 目的
Googleにサイトの存在を通知し、検索結果に表示されるようにする。

### ✨ 期待される効果
- **検索結果に表示される**（最重要！）
- 検索クエリの把握
- 検索順位の確認
- クリック率の確認
- 技術的問題の早期発見

### 📝 登録手順

**Step 1:** https://search.google.com/search-console にアクセス

**Step 2:** 「プロパティを追加」→「URLプレフィックス」を選択

**Step 3:** URLを入力: `https://prime-ministers-app.vercel.app/`

**Step 4:** 所有権の確認（HTMLタグ推奨）
```html
<meta name="google-site-verification" content="XXXXXX" />
```
を index.html の <head> 内に追加

**Step 5:** サイトマップを送信
- 左メニュー「サイトマップ」→「sitemap.xml」と入力して送信

### 📊 活用方法

**検索パフォーマンス（重要）**
```
確認項目:
- クリック数
- 表示回数
- 平均CTR
- 平均掲載順位
- 検索クエリ（どんな言葉で検索されているか）

活用法:
- クリック率が低いページの改善
- 掲載順位が低いキーワードの強化
```

**カバレッジ**
```
確認項目:
- インデックス登録済みページ数
- エラーの有無

活用法:
- 全ページがインデックスされているか確認
- エラーがあれば即座に修正
```

### 📈 効果測定タイムライン
```
1週間後: インデックス開始
1ヶ月後: 検索結果に表示開始
3ヶ月後: 安定した検索流入確立
```

## 📊 3. Google Analytics 設定

### 🎯 目的
アクセス解析を行い、ユーザーの行動を理解する。

### ✨ 期待される効果
- アクセス数の把握（日別・月別）
- ユーザー属性の理解
- 人気ページの特定
- 流入経路の分析
- コンバージョンの測定

### 📝 設定手順

**Step 1:** https://analytics.google.com/ にアクセス

**Step 2:** アカウント作成
```
アカウント名: 首相マスター
プロパティ名: 首相マスターアプリ
タイムゾーン: 日本
通貨: 日本円（JPY）
業種: 教育
```

**Step 3:** データストリーム設定
```
プラットフォーム: ウェブ
URL: https://prime-ministers-app.vercel.app
ストリーム名: 首相マスター本番環境
```

**Step 4:** 測定ID（G-XXXXXXXXXX）を取得

**Step 5:** index.html に追加
```html
<!-- Google Analytics -->
<script async src="https://www.googletagmanager.com/gtag/js?id=G-XXXXXXXXXX"></script>
<script>
  window.dataLayer = window.dataLayer || [];
  function gtag(){dataLayer.push(arguments);}
  gtag('js', new Date());
  gtag('config', 'G-XXXXXXXXXX');
</script>
```

### 📊 主要レポートの見方

**リアルタイム**
- 現在のアクティブユーザー数
- 閲覧中のページ
- 流入元

**ユーザー属性**
- 新規 vs リピーター
- 国・地域
- デバイス（PC・スマホ）

**集客レポート**
- Organic Search（検索エンジン）
- Direct（直接アクセス）
- Referral（他サイトから）
- Social（SNS）

**エンゲージメント**
- 人気ページ
- 平均滞在時間
- 離脱率

### 📈 効果測定タイムライン
```
1週間後: データ蓄積開始
1ヶ月後: トレンドが見えてくる
3ヶ月後: データに基づく意思決定が可能
```
