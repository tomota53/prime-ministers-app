### 10. 口コミ・紹介制度 ★★★★☆

#### バイラルマーケティング

**紹介機能の実装:**
```jsx
// シェアボタンを追加
<button onClick={handleShare}>
  <Share size={20} />
  友達に紹介する
</button>

const handleShare = () => {
  if (navigator.share) {
    navigator.share({
      title: '首相マスター',
      text: '中学受験に使える無料アプリ！',
      url: window.location.href
    });
  }
};
```

**紹介キャンペーン:**
```
「友達紹介で特典GET！」

紹介した人: プレミアム機能1ヶ月無料
紹介された人: 追加問題集プレゼント
```
